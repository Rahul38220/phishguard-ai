# PhishGuard AI: Multilingual Phishing Detection Framework
**Lead Researcher & Developer** | Published in IRJET (Oct 2025)

## 📌 Overview
PhishGuard AI is an advanced cybersecurity framework designed to detect phishing attempts across multiple languages (English, Hindi, and Gujarati). Unlike standard filters, this model provides real-time "reasoning" for flags, empowering users with digital literacy in their native language.

## 🚀 Key Features
- **Multilingual Detection:** Powered by XLM-RoBERTa & mT5 to handle code-switching and regional dialects.
- **Explainable AI (XAI):** Provides human-readable explanations of why a specific link or email was flagged.
- **High Accuracy:** Optimized for low false-positive rates in SMS (Smishing) and Email datasets.

## 🛠 Tech Stack
- **Languages:** Python (HuggingFace, PyTorch)
- **Models:** XLM-RoBERTa, mT5
- **Dataset:** Custom-curated multilingual phishing corpus
